GitHub repository link: https://github.com/Maojia-Yao/AI-Coder.git


User Instruction:

1.
Install python3.6 or above version

2.
Create a virtual environment under the terminal in the project:
For Mac or Linux system: python3 -m venv venv
For Windows system: python -m venv venv

3.
Activate virtual environment:
For Mac or Linux system: source venv/bin/activate
For Windows system: venv\Scripts\activate

4.
Install all dependencies listed in the requirements.txt file in the virtual environment:
pip install -r requirements.txt

5.
Run the website:
python manage.py runserver

6.
Open the browser and access the following URL:
http://127.0.0.1:8000/

7.
If you want to view and use the coding, explaining, evaluation and profile pages, you must register and log in.
